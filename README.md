# qualifying offer

Scrapes players and the salaries to calculate the qualifying offer based on the top 150 highest salaries.

## Prerequisites
- Java 8+
- Maven 3.3.x+
- Git

## Setup and Run Steps
1. `git clone git@github.com:thespags/phillies.git`
2. `cd phillies`
3. `mvn package`
4. `java -jar target/offer-1.0-SNAPSHOT.jar`
